Ei(x i)=(x i+K i) % 26

  • We have to approximately the same formula as we used for Caesar Cipher
  • x i is the actual letter in the plaintext
  • Ei(xi) is the encrypted letter in the ciphertext
  • in Vigenere cipher we have to use the i-th letter of the key for encrypting the i-th letter
why to use mod 26?

The size of the english alphabet is 26 which means there are 26 lettes in the english alphabet

==> we want to make sure the encrypted letter is within the range[0,SIZE_ALPHABET-1] so this is why to use 26 here
